## How to install

1. On the Android phone you want to install on, tap **Download APK** above.
2. Open the downloaded `.apk` file.
3. If prompted, allow installation from **unknown sources** (Settings → Apps → Special access).
4. Follow the on-screen prompts to finish installing.
